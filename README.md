# Scott-app

## How to run?
1. Clone repo `git clone https://github.com/andbrew/Scott-app.git`
2. Load Scott schema (`scott.sql`)
3. Change `user`, `psswd`, `host` and `SID` for your owns in `app.py`.
4. Run `app.py`
